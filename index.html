Teacher Notes

This is a single-file HTML game. Students can play independently on Chromebooks. It does not collect names, log data, use ads, or require a login.

Best classroom use: give students scratch paper or whiteboards. Tell them the game is fun, but the math answer must be worked out before clicking.

To change questions, scroll to the questions array in the code and edit the text, answer choices, correct index, or explanation.
